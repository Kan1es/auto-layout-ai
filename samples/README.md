# Тестовые изображения

Эта папка предназначена для небольших локальных изображений, на которых удобно
проверять DART без загрузки ZIP-датасета через интерфейс.

После установки основного окружения выполните из корня репозитория:

```powershell
.\.venv\Scripts\python.exe scripts/create_sample_image.py
```

Команда создаст `samples/test.jpg`. Это техническая тестовая картинка: DART может
вернуть для неё пустой результат, но запуск подтвердит, что модель, зависимости и
пути настроены правильно. Для содержательной проверки замените `test.jpg` своей
фотографией и укажите подходящий текстовый prompt.

Файлы изображений из этой папки не добавляются в Git.
